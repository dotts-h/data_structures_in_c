# Lab_4 - Doubly Linked Lists

## Operatii Principale - Main Operations

> Fie o lista dublu inlantuita in cardul careia informatia utila reprezinta un numar natural (un anumit numar poate sa apara de mai multe ori). Sa se implementeze urmatoarele operatii de baza ce pot fi selectate prin intermediul unui meniu principal si mai multe sub-meniuri
> Let there be a doubly linked list which can store natural numbers (allows duplicates). > Implement the following basic operations which can be selected via a main meniu and sub-menus.

- Add
  - At the beginning
  - At the end
  - Before an element identified by value(key)
  - After an element identified by value(key)
- Delete
  - First
  - Last
  - An element identified by value(key)
- Search
  - By value(key)
- Print
  - Start to End
  - End to Start

## Operatii Extinse - Extra Operations

> Sa se adauge optiunea "Operatii extinse" la meniul principal ce va permite urmatoarele:
> Add to the main menu the option "Extra Operations" which will provide the following functionalities

- Remove duplicates
- Add a number after each of its divisor
- Sort Ascending
- Sort Descending
